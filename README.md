# Bits-in-Bio CPH Hackathon

**Theme: Unlocking Data Silos**

## Challenge
What bio tools can you build in 2.5 hours!?

## Concept
We have much diverse talent in BiB, so we thought it would be fun to create a hackathon to inspire what is possible with today's technology to (vibe)code software to answer biological questions - for just yourself, your team, or your organization to help with data entry, literature search, visualizations, experiment planning and much more...

We will provide mentoring and inspiration to get you started. Afterwards out start working in small teams in a relaxed atmosphere on something that excites you.

## You bring
Own laptop + Claude/Codex/Lovable... set up.

## Bits in Bio
Bits in Bio is a global community building software for science. We welcome software people interested in solving problems in biology, bio people wanting to use software in the lab, and everybody in between. We look forward to seeing you!

## Our Project: Viably

**Unlocking data silos through a unified decision brief.**

Viably helps researchers and founders decide whether a bio project idea is worth pursuing, reviewing carefully, or reframing by checking structured evidence across studies, trials, interventions, and related records.

Built with the [Amass API](https://platform.amass.tech), which unifies biomedical data across six cores into one interface.
